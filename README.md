# CRUD em JSON - Projeto de Backend

## 📌 Descrição do Projeto
Este projeto é um sistema simples de backend para gerenciamento de dados utilizando **arquivo JSON** como base de dados, sem uso de bancos de dados ou frameworks.  
O sistema permite **Criar, Ler, Atualizar e Excluir (CRUD)** registros organizados por grupos, como `alunos` e `professores`.

---

## ⚙️ Funcionalidades

### Create (Criar)
- Adicionar um novo registro a um grupo do JSON (`alunos` ou `professores`).

### Read (Ler)
- Listar todos os registros.
- Listar registros por grupo.

### Update (Atualizar)
- Atualizar dados de um registro específico usando seu `id`.

### Delete (Excluir)
- Remover um registro do JSON usando seu `id`.

---

## 🛠️ Tecnologias Utilizadas
- Linguagem: **Python 3**
- Módulos: `json`, `os`
- Execução: via **terminal**
- Persistência: **arquivo JSON**
